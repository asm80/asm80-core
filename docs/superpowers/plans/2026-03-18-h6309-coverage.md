## Background for implementer

`cpu/h6309.js` exports `H6309` — an HD6309 assembler that extends M6809. Its `parseOpcode(s, vars, Parser)` function handles indexed addressing modes. The currently untested paths are all inside the indexed addressing logic:

- **5-bit offset** (small n, −16 < n < 16, non-PC, non-indirect): postbyte encodes offset directly, no extra bytes
- **8-bit offset** (−128 < n < 128, outside 5-bit): postbyte `| 0x88` + 1 offset byte
- **16-bit offset** (large n, or W register): postbyte `| 0x89` + 2 bytes; W always uses postbyte `0xAF`
- **null offset fallthrough**: when expression is undefined, must use PC as base to bypass 5-bit guard
- **indirect extended** `[addr]`: single param starting with `[`, postbyte `0x9F`
- **AIM extended/indexed**: existing tests only cover AIM direct
- **Error paths**: bad registers, wrong param counts

**Key constants:**
- `vars = { _PC: 0x0100 }` (declared at top of existing test file, reused by all modules)
- `LDA` indexed opcode = `0xA6` (1-byte); `LDW` indexed = `0x10A6` (2-byte prefix)
- `AIM` direct/indexed/extended = `0x02` / `0x62` / `0x72`

**Indexed mode params format:**
- Normal: `params: ["offset", "register"]` → e.g., `["100", "X"]`
- Indirect: `params: ["[offset", "register]"]` → e.g., `["[100", "X]"]`
- Indirect extended: `params: ["[$1000]"]` (single param)

**W register restriction:** The 8-bit path calls `ixregPC(p2)` which throws on `"W"`. W indexed must use offset ≥ 128 to bypass the 8-bit path and reach the 16-bit path which has a W guard.

**Spec:** `docs/superpowers/specs/2026-03-18-h6309-coverage-design.md`

QUnit.module("H6309 indexed — null offset fallthrough");

QUnit.test("LDA UNDEF,PC — undefined symbol falls through to 16-bit (zptest===null)", function () {
  // Parser.evaluate("UNDEF", vars) throws → zptest=null.
  // 5-bit gate: ixregPC("PC")===4 → condition false, 5-bit path skipped.
  // 8-bit gate: zptest!==null is false → 8-bit path skipped.
  // Falls to 16-bit block: s.bytes+=2 → 4. W guard: "P"!=="W" → ixregPC("PC")|0x89 = 0x8d.
  var s = { opcode: "LDA", params: ["UNDEF", "PC"], addr: 0x100, lens: [], bytes: 0 };
  var p = H6309.parseOpcode(s, vars, Parser);
  QUnit.assert.equal(p.bytes, 4, "bytes — null zptest falls through to 16-bit");
  QUnit.assert.equal(p.lens[1], 0x8d, "postbyte 0x8d (PC 16-bit)");
  QUnit.assert.equal(p.lens[3], null, "null terminator");
});

QUnit.module("H6309 indexed — indirect extended");

QUnit.test("LDA [$1000] — indirect extended 1-byte opcode", function () {
  // Single param starting with "[": indirect extended path.
  // ax[2]=0xa6 ≤ 256: lens[0]=0xa6, postbyte=1. lens[1]=0x9f, lens[2]=func, lens[3]=null.
